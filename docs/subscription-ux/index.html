Subscription creation — 4 UX options

Prototypes that simplify the current two-big-cards chooser. Each is self-contained (mock data, no backend) so you can click through and validate. Open each, then tell me which one to ship.

Option 1

Segmented toggle

A compact User / Group switch at the top with one inline form below. Everything stays on a single, small screen — no separate panels.

 Option 2

Progressive disclosure

One simple form defaulting to a personal subscription. A “Share with a group” switch reveals the group picker only when needed.

 Option 3

Dropdown selector

The most minimal: a type dropdown + name field + Subscribe button in one tidy row. The group field appears only for group subscriptions.

 Option 4

Compact icon cards

The familiar cards, but smaller, with icons, and they expand inline (accordion) instead of replacing the whole view.
