Group Subscriptions — Collapsible filter drawer

Option 4. A persistent search box plus a "Filters" button (with a badge counting how many filters are active) that opens a side drawer. The drawer groups every facet — Type, State, Product — each option showing a live count, with Apply and Clear. Best when the widget is narrow and toolbar space is scarce. Active filters appear as removable chips above the grid; the Actions column stays pinned.
